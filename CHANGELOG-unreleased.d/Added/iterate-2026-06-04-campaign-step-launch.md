Campaigns lane: the per-step action is now a one-click **Launch (Cx)** button that opens a terminal running `/shipwright-iterate "<specPath>"` for the campaign's next-pending sub-iterate (replacing the old "Copy launch" clipboard button). An ordinary step launches directly; a confirm dialog appears only when that step previously failed/escalated or is plan-first. The command is built entirely server-side from the validated `{slug, stepId}` — the client never sends the path.
