Per-run data join: read-only `GET /api/external/projects/:projectId/{runs,runs/:runId,grade-trend}` endpoints (plus `lib/runDataApi.ts` + `useRunData` hooks) that turn the tracked event log into the per-run shape the Mission-Control / Ship's-Log / board surfaces consume — a run joined to its affected FRs, test counts, DERIVED gate lamps (flagged as derived, never an authoritative verdict), iterate phase-timings, and a grade-trend series folded from `grade_snapshot` events. Honest degradation throughout: an unknown run or a run without timings returns `null` (rendered `n/a`) — durations are never fabricated (FR-01.47).
