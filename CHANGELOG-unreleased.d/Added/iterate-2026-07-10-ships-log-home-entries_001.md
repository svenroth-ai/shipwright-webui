A project's home is now its Ship's Log. Opening a project from the Projects gallery lands on a new logbook page: a slim "Captain's Drawer" strip that shows the project's control grade with a small ring and honest sub-scores (and a "Why an A?" link that opens the real compliance record — nothing invented), a logbook of every recorded run as a clickable entry into its mission (a run with no linked mission is shown but not a dead click), and a "What do you want to change?" box that turns a one-line brief into a scoped plan and launches the next iterate with a single Go — the same safe launch path as everywhere else. A project with no recorded runs shows an honest empty logbook instead of fake rows. Board task cards stay lean: no FR tags, no gate dots (that detail lives in the logbook and task detail now).
