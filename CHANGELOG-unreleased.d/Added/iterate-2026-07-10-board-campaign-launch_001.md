Launching a campaign or a step is now visible and recoverable from start to finish. A planned (draft) campaign shows up on the board with a status badge and a "Start campaign" button, so you no longer have to open Triage to begin it. Every launch first shows a plain confirmation — what will run, in which project, and the exact command — and an autonomous run also lists the remaining steps and warns it won't ask again; Cancel does nothing. Most importantly, a launch that fails is no longer silent: whether the command never ran, no transcript appeared, the folder can't be written to, or the campaign already finished, you get a clear message that stays put (never a toast that vanishes when you switch tabs) with the right way forward — a Retry only when retrying can actually help, otherwise the fix stated plainly. The same wording appears on the campaign card, the board task card, and the task page, so a failure reads the same everywhere.
