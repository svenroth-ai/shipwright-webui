The Projects page is now a gallery of Ship's-Log preview cards instead of a plain table. Each project is a card showing its colour, name, compliance grade, and — the point of the change — a preview of its logbook: projects that have actually been run show a small runs sparkline, a "N runs · M FRs · score/100" line, and a quote from the most recent run's summary; projects with no runs yet show an honest "No runs yet — grade it to open the logbook." sentence instead of a fake, empty chart. Every number is read from real run data — nothing is invented, and while the data is still loading (or if it can't be read) the card says so rather than falsely claiming "no runs". Projects that have a logbook sort to the front, so a board with many projects leads with the ones that have real history. The gear (settings) and trash (remove, with its task-cascade warning) work exactly as before. "Open board" opens that project on the task board, and an ungraded project's "Grade it" opens the read-only Grade tool — the Command Center still never runs anything on its own.
