Mission Control "The Record" rail: one clickable spine that merges the process rail and the trace chain into a single artifact (absorbing the former TraceChain) — five state-derived nodes (Requirement · Spec · Tests · Review · Commit), each carrying its receipt, that open an artifact panel (an inline card on the photo, a scrimmed slide-over on narrow screens) whose "Open full document" routes to the existing Files & Terminal viewer. Node state and every receipt are derived honestly from real per-run data (FR-01.47) — a step with no evidence reads "pending" with no receipt, a held review never shows "clean", a run with no commit never invents one, and a design gate shows the Design step as current. Adds the three Grade / Tests / Serves instrument chips (live values with run data, an honest "—" without) and the shared `useMissionState` derivation the rest of Mission Control consumes. The task detail gains a Mission | Files & Terminal switch; the terminal stays the default view (FR-01.55).
