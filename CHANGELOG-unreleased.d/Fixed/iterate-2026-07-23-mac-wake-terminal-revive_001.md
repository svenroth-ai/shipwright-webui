Embedded terminal (macOS): recover in a few seconds after the Mac sleeps, instead of sitting frozen ~30-60 seconds or needing a browser reload. The previous fix's fast reconnect only ran when the browser signalled focus/visibility/network changes on wake — which macOS does not fire when the whole machine sleeps with the window still in front — so recovery fell back to a slow ~45-second heartbeat. A wake detector now notices the machine was asleep (by the jump in elapsed time) and re-checks the connection immediately, on any operating system. (FR-01.28)
