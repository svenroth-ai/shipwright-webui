Right-click in the terminal no longer pastes twice. Claude Code treats a right-click as a paste (from its own copy buffer), and the embedded terminal was forwarding the right-click to it — so a right-click pasted once *via Claude* **and** once via the browser's own context-menu "Paste". The terminal now no longer forwards right-button mouse events to Claude, so a right-click is browser-only (menu → Paste) = exactly one paste. Left/middle-click and the scroll wheel are unchanged — Claude's selection, clicks and scrolling still work.
