- The Preview button's opened URL is now host-pinned to your local dev server (D21/F10): a hand-authored project profile whose `dev_server.ready_path` smuggled another host (e.g. `@evil.com/`) could previously redirect the opened preview off your machine — the returned URL is now built through the same localhost-pinning sanitizer the readiness probe uses, so a malicious or malformed path can only ever land on your own dev server (or its root), never off-host.
