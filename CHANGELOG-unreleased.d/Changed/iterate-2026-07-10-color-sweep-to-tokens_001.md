Every hardcoded Tailwind palette class (`bg-stone-100`, `text-red-600`, …) and arbitrary-hex colour class (`bg-[#5c5652]`) across the app is swept onto the Weather-Deck design tokens: the legacy warm-beige `--color-*` names now alias A03's token system (one edit re-skins 368 call sites), the sidebar is anthracite, and primary actions read teal. A guard test (`no-hardcoded-colors.test.ts`) fails the build if a palette or hex colour class is reintroduced.
